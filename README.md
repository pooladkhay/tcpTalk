# tcpTalk

### tldr;
Run Server, Run Client and boom!
You will hear client's mic on the server :D

## Requirements
This project uses [PortAudio](http://www.portaudio.com) as Audio I/O library.

### Install PortAudio

macOS:
```brew install portaudio```

Ubuntu:
```sudo apt install portaudio19-dev```
